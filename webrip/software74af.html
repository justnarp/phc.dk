SAMBA

Version 2005.7

Introduction

SAMBA is a Windows version of the fast and powerful combined sewer planning tool used by professionals and researchers for almost 30 years. The simple, well founded and well understood calculation core of the SAMBA model has only undergone minor changes compared to earlier releases. The user-friendly windowed interface is not the only feature that makes this new SAMBA standout from earlier versions and similar tools. A variety of added user facilities including system editing directly in the simplified system format makes SAMBA a fast and powerful planning tool. Its functionality makes it ideal for evaluating alternative scenarios, optimising system design and performance and optimising investments in the integrated urban water environment.

SAMBA has also been implemented as a subset of components to www.WaterAspects.org. If you require flexibility in your work we recommend that you take a look at the new possibilities that this presents. SAMBA within the WaterAspects environment is significantly slower than the traditional SAMBA. Therefore, if calculation speed is important for you, we recommend that you stick to this traditional SAMBA in so far as it has the features you need.

Note that this Windows SAMBA requires Mike Urban or Mouse from DHI Software to create the sewer network and catchment input files (MEX or UND/HGF or SWF). In WaterAspects you have the alternative of either defining the simplified network directly yourself or importing and simplifying a Mike Urban/Mouse network.

Installation

The file is self-extracting. Just double-click and run the installation directly.
